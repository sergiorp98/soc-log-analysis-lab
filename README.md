# SOC Log Analysis - Apache

## Objective
Simulate a SOC investigation detecting suspicious activity in Apache access logs.

## Scenario
A web server shows abnormal traffic patterns. The goal is to identify:
- Brute force attempts
- Suspicious IP behavior
- Unusual request patterns

## Tools Used
- Phython
- Manual log analysis
- Basic pattern detection

## Skills Demonstrated
- Log analysis
- Threat detection
- Documentation of investigation
- Basic scripting for automation
